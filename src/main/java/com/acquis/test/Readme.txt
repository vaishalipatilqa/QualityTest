
Implemented test cases using the REST Assured library and TestNG.

Created automated test cases to verify the User CRUD operations using the REST APIs available on 
here https://reqres.in/.

Ensured comprehensive test coverage for RESTful operations on the following endpoints:
	/api/users
	/api/users/{userId}

-------------------------------------------------------------------------------------------------

there are two classes 
	1. CRUDClass - Performed basic operations GET, POST, PUT, DLETE
	2. UserOperations - Performed various operation on Users resourse (single user, specific user, list of users etc)
	3. Payload class - to keep body JSON file seperately.
	4. README file - how to execute or the explanations of project.
